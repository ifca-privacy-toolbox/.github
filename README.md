# IFCA Privacy Toolbox

Welcome to the IFCA Privacy Toolbox organization. This space brings together research tools developed within by the [IFCA Advanced Computing and e-Science Group](https://github.com/IFCA-Advanced-Computing), focusing on privacy-preserving data analysis and differential privacy techniques. This organization provides the landing page for the toolbox.

---

## Three tools, three objectives

### anjana
[![Repo](https://github-readme-stats.vercel.app/api/pin/?username=IFCA-Advanced-Computing&repo=anjana)](https://github.com/IFCA-Advanced-Computing/anjana)

Apply nine anonymization techniques, from k-anonymity, l-diversity, t-closeness to β-likeness and δ-disclosure privacy among others to tabular datasets. Designed for privacy-compliant data sharing in research and industry.

Interactive web application (powered by EOSC SIESTA): [https://anjana.cloud.eosc-siesta.eu/](https://anjana.cloud.eosc-siesta.eu/)

---

### pyCANON
[![Repo](https://github-readme-stats.vercel.app/api/pin/?username=IFCA-Advanced-Computing&repo=pycanon)](https://github.com/IFCA-Advanced-Computing/pycanon)

Check and measure the level of anonymization of a tabular dataset. Compute the parameters related to the nine anonymziation techniques implemented in anjana. In addition, provides metrics for checking the privacy-utiloty trde-offs and the risks of re-identification and disclousre. It allows to audit your data before release.

Interactive web application (powered by EOSC SIESTA): [https://pycanon.cloud.eosc-siesta.eu/](https://pycanon.cloud.eosc-siesta.eu/)

---

### trasgoDP
[![Repo](https://github-readme-stats.vercel.app/api/pin/?username=IFCA-Advanced-Computing&repo=trasgoDP)](https://github.com/IFCA-Advanced-Computing/trasgoDP)

Apply local based-differential privacy mechanisms directlly to tabular, both numerical, categorical and location-based data (via geo-indistiguisability). It implementes a correlation loss metric and allwos to check the divergence. Designed for realeasing sanitized versions of the data with the privacy guarantees provided by the privacy budget.

Interactive web application (powered by EOSC SIESTA): [https://trasgodp.cloud.eosc-siesta.eu/](https://trasgodp.cloud.eosc-siesta.eu/)
